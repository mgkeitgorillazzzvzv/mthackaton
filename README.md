# Хакатон колледжей Москвы: Московский Транспорт
Сайт для хакатона 

# Установка и запуск

1. `git clone https://github.com/mgkeitgorillazzzvzv/drivehack.git && cd drivehack`
2. Настройте `.env` файл на основе `.env.example`
3. `pnpm i`
4. `pnpm run build`
5. `pnpm run db:push`

Для запуска сервера:

`node build/index.js`
